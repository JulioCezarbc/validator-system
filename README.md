# Validator System

Sistema de validações desenvolvido para estudo utilizando frontend e backend separados em um único repositório.

## 🚀 Tecnologias

### Frontend
- Vue 3
- Nuxt 4
- TypeScript
- Nuxt UI
- Axios

### Backend
- Node.js
- Express
- TypeScript

---

## 📁 Estrutura do Projeto

```
validator-system/
├── frontend/
└── backend/
```

---

## ✅ Funcionalidades

- Validação de CNPJ
- Validação Mod10
- Validação Luhn
- Interface moderna e responsiva
- Integração frontend/backend

---

## ⚙️ Como executar

### Clone o projeto

```bash
git clone https://github.com/JulioCezarbc/validator-system.git
```

---

# Backend

```bash
cd backend
npm install
npm run dev
```

Servidor disponível em:

```
http://localhost:3000
```

---

# Frontend

```bash
cd frontend
npm install
npm run dev
```

Aplicação disponível em:

```
http://localhost:3001
```
